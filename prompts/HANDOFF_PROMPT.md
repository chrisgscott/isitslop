## Architecture Summary
- **Frontend:** Next.js 15 + Tailwind on Vercel (isitslop.co)
- **Analysis Service:** Python on Modal (serverless webhook, same pattern as SanityCheck)
- **Database:** Supabase (analyses table, no auth, no RLS)
- **AI:** OpenAI GPT-4.1-mini for verdict writing only — scoring is deterministic
- **Flow:** User pastes URL → API creates pending record + fires Modal webhook → Modal downloads tarball, analyzes, scores, writes verdict, saves to Supabase → Frontend polls and renders result

## Key Constraints
- No auth — zero friction, paste-and-go
- No ShadCN — custom Tailwind for distinctive design, use frontend-design skill
- Deterministic scoring — AI writes verdicts, doesn't influence scores
- Results are public — shareable URLs are the growth mechanism
- Tone: "your AI did you dirty — here are the receipts"
