# Project Brief: IsItSlop

## Executive Summary
IsItSlop is a vibe code gut check. Paste a GitHub repo URL, get a Slop Score (0-100) with letter grades across six dimensions, a snarky verdict with specific receipts, and copy-paste fix prompts you can feed back to your AI tool. The growth loop is shareability — every result has a static URL designed to be dropped in PRs, Slack threads, and Twitter.

## The Problem
Developers are vibe coding with AI tools (Claude, Cursor, Copilot) and shipping code they haven't fully reviewed. They know it might be a mess but have no quick way to find out. Existing code quality tools (SonarQube, CodeClimate) are enterprise-grade, require CI/CD integration, and produce clinical reports nobody shares.

## The Solution
A zero-friction web app: paste a GitHub URL, get a brutally honest assessment in 30-60 seconds. The analysis is deterministic (file system scanning, regex patterns, package.json inspection). The verdict is AI-generated — snarky, specific, and actionable. Each finding comes with a ready-to-paste prompt so you can tell your AI to fix its own mess.

## Target Users
Developers who use AI coding tools and want a sanity check on what their AI produced. They know they vibe coded it. They want to know how bad it is and what to fix.

## Key Features (MVP)
1. URL input → repo analysis → Slop Score with 6 dimension grades
2. Snarky, shareable verdict with specific receipts per finding
3. Copy-paste fix prompts for each issue ("give this to Claude and tell him...")
4. Shareable static result URLs
5. Distinctive result page that looks like a product, not a dashboard

## Success Criteria
- People voluntarily share result URLs on social media / in PRs
- Analysis completes in under 60 seconds for typical repos
- Verdicts are genuinely funny and specific, not generic

## Current Status
- **Phase:** Planning Complete
- **Next Step:** Begin Phase 1 (Foundation)

## Technical Approach
- **Frontend:** Next.js 15 (App Router) + Tailwind on Vercel
- **Analysis Service:** Python on Modal (serverless, same pattern as SanityCheck)
- **Database:** Supabase (store results, power shareable URLs)
- **AI:** OpenAI GPT-4.1-mini for verdict generation
- **No auth for MVP** — public repos only

## Out of Scope (for MVP)
- Private repo analysis (requires auth + GitHub OAuth)
- Badge endpoint for README embeds
- MCP server for CLI access
- Paid tier / freemium model
- ESLint/semgrep integration
- Multi-language AST parsing
